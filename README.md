# ubuntu-binary-translator

A modern binary conversion application designed to translate binary values into human-readable formats with an intuitive interface and efficient conversion logic.

---

## Features

- Binary to Decimal conversion
- Decimal to Binary conversion
- Binary to ASCII text conversion
- ASCII to Binary conversion
- Binary to Hexadecimal conversion
- Hexadecimal to Binary conversion
- Input validation and error handling
- Fast and accurate translation
- Clean and user-friendly interface

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Omkarbahirat/binary-translator.git
cd binary-translator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Run Application

Run the application:

```bash
python main.py
```

If using Python 3 explicitly:

```bash
python3 main.py
```
